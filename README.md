# TMDb
Thoughtworks Movie Database

A movie Gallery App

### Get the code

If you don't have the code yet grab it by:

`git clone https://github.com/evilc0des/tmdb.git`

### For Development

Run these commands:
`npm install`
<br>
`npm run dev -s` 

### For Deployment

First Build the app:
`npm install`
`npm run build` 

Then start server:
`npm start`

Open the app at http://localhost:8080 or go to the address shown in the console
